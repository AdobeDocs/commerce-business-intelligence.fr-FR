# Créer une copie d’un rapport

Souvent, vous souhaiterez créer des graphiques définis de manière similaire dans lesquels vous ne souhaitez modifier qu’un seul élément, comme la modification d’un filtre ou la modification de la variable `group by`.

Dans ce cas, lancez la variable `Chart Editor` et cliquez sur **[!UICONTROL Save As]** dans le coin supérieur droit. Cela répliquera votre graphique existant et l’ajoutera à votre tableau de bord actuel avec le nouveau nom que vous avez sélectionné. Cela vous permettra de modifier immédiatement les paramètres du nouveau graphique.
